# 奇幻世界基础规则模板

## 世界设定

- **时代背景**: [中世纪/蒸汽朋克/现代奇幻等]
- **魔法系统**: [描述魔法如何运作]
- **主要种族**: [人类/精灵/矮人/兽人等]

## 角色规则
### 角色创建
1. 每个角色必须有明确的背景故事
2. 角色能力应符合世界设定
3. 角色动机应清晰明确

### 角色发展
1. 角色可以通过经验学习新技能
2. 角色关系会随着互动发展
3. 重大事件可能改变角色性格

## 魔法规则
### 魔法类型
- **元素魔法**: 操控自然元素
- **奥术魔法**: 纯粹的魔法能量
- **治疗魔法**: 治愈和恢复
- **召唤魔法**: 召唤生物和仆从

### 魔法限制
1. 魔法需要消耗魔力或材料
2. 强大魔法需要吟唱时间
3. 魔法反噬可能发生

## 战斗规则
### 战斗流程
1. 先攻判定
2. 行动轮次
3. 伤害计算
4. 状态效果

### 伤害类型
- **物理伤害**: 剑、箭等武器造成
- **魔法伤害**: 法术造成
- **精神伤害**: 精神攻击造成

## 叙事规则
### 一致性要求
1. 已发生的事件不可随意更改
2. 角色行为应符合其性格
3. 世界规则应保持一致

### 玩家干预
1. 玩家可以提出建议
2. 重大决定需要玩家确认
3. 玩家可以要求重述

## 特殊规则
### 时间旅行
- 仅限特定情节允许
- 必须符合因果逻辑
- 可能产生时间悖论

### 平行世界
- 可以探索不同可能性
- 需要明确区分时间线
- 角色记忆可能受影响

## 示例规则片段
```yaml
rule: "魔法消耗"
description: "施展魔法需要消耗魔力"
condition: "当角色施展魔法时"
effect: "角色的魔力值减少相应数值"
exception: "某些特殊物品可能提供无限魔力"
```

## 使用说明
1. 复制此模板并根据需要修改
2. 填充方括号中的内容
3. 添加自定义规则
4. 测试规则一致性
