# 科幻世界基础规则模板

## 世界设定
- **世界名称**: [在此输入世界名称]
- **时代背景**: [近未来/星际时代/后末日等]
- **科技水平**: [描述主要科技]
- **主要势力**: [描述星际势力或国家]

## 角色规则
### 角色创建
1. 角色必须有专业背景（科学家、飞行员、工程师等）
2. 角色技能应符合科技设定
3. 角色可能有机械或生物增强

### 角色发展
1. 角色可以通过训练提升技能
2. 科技升级可能改变角色能力
3. 星际旅行可能影响角色心理

## 科技规则
### 科技类型
- **太空旅行**: 超光速引擎、虫洞技术
- **人工智能**: 智能助手、自主机器人
- **生物科技**: 基因改造、克隆技术
- **武器系统**: 激光武器、等离子炮

### 科技限制
1. 高级科技需要稀有资源
2. 科技故障可能发生
3. 某些科技可能有伦理限制

## 太空探索规则
### 飞船系统
1. 生命支持系统
2. 推进系统
3. 武器和防御系统
4. 扫描和通讯系统

### 星际旅行
1. 燃料消耗计算
2. 航行时间估算
3. 空间异常处理
4. 外星文明接触协议

## 叙事规则
### 科学准确性
1. 物理定律应基本遵守
2. 科技发展应有合理逻辑
3. 外星生物应符合进化原理

### 玩家干预
1. 玩家可以提出科学建议
2. 技术细节可以协商
3. 玩家可以要求科学解释

## 特殊规则
### 时间膨胀
- 接近光速旅行时发生
- 影响角色年龄和感知
- 可能产生时间悖论

### 平行宇宙
- 量子理论允许的可能性
- 需要明确宇宙编号
- 角色可能跨宇宙旅行

## 示例规则片段
```yaml
rule: "飞船燃料消耗"
description: "超光速旅行消耗反物质燃料"
condition: "当飞船进行超光速跳跃时"
effect: "飞船燃料减少相应数值"
exception: "某些星系的特殊能量场可能减少消耗"
```

## 使用说明
1. 复制此模板并根据需要修改
2. 填充方括号中的内容
3. 添加自定义科技规则
4. 测试规则的科学一致性
