# LOOM v0.10.0 发布说明

## 概述

LOOM v0.10.0 是项目重构的**阶段1发布版本**，标志着项目定位重构的完成和发布准备工作的就绪。此版本专注于系统集成、接口标准化和发布基础设施的完善。

**发布日期**: 2026-01-12
**版本类型**: 功能发布（阶段1完成）
**兼容性**: 向后兼容 v0.9.x（如存在），但建议全新安装

## 主要变更

### 1. 项目定位重构完成
- **术语更新**: 从"游戏引擎"改为"叙事解释器"相关术语
- **架构清晰化**: 明确五层架构（核心、规则、解释、记忆、干预）
- **文档一致性**: 更新所有文档、API和用户界面中的术语

### 2. 系统集成验证
- **五层架构集成**: 验证核心运行时、规则层、解释层、记忆层、干预层之间的接口兼容性
- **组件测试**: 完成87个集成测试中的68个（78%通过率）
- **接口标准化**: 统一各层之间的数据交换格式和API约定

### 3. 发布基础设施
- **版本管理**: 更新到语义化版本 v0.10.0
- **打包配置**: 完善 `pyproject.toml` 配置
- **发布检查清单**: 创建完整的发布流程和质量检查清单

### 4. 部署配置更新
- **Docker配置**: 支持新功能的多阶段构建
- **Kubernetes配置**: 更新部署和服务配置
- **环境变量**: 标准化环境变量配置

## 技术细节

### 架构改进
- **核心运行时**: 会话管理、回合调度、持久化引擎优化
- **规则层**: Markdown规则解析和版本控制增强
- **解释层**: LLM推理流水线一致性检查改进
- **记忆层**: 结构化存储和向量存储集成
- **干预层**: OOC处理、世界编辑、Retcon处理标准化

### 接口变更
- **向后兼容**: 保持主要API的向后兼容性
- **新增接口**: 添加组件集成接口
- **错误处理**: 改进错误处理和异常管理

### 性能优化
- **异步处理**: 优化高并发会话处理
- **内存管理**: 改进记忆存储和检索效率
- **响应时间**: 减少解释层响应延迟

## 安装和升级

### 新安装
```bash
pip install loom==0.10.0
```

### 从 v0.9.x 升级
1. 备份现有配置和数据
2. 查看迁移指南 (`docs/UPGRADE_GUIDE.md`)
3. 执行升级：
   ```bash
   pip install --upgrade loom==0.10.0
   ```
4. 验证安装：`loom --version`

### Docker 部署
```bash
docker pull loom:0.10.0
docker-compose up -d
```
