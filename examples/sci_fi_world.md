# 科幻世界示例：星海联邦
# 版本: 2.0.0
# 作者: LOOM科幻团队
# 类型: 太空歌剧、硬科幻元素

# 世界观 (World)

## 宇宙概述
公元25世纪，人类已建立横跨数百光年的"星海联邦"。联邦由核心世界、边缘殖民地和无数空间站组成，通过跃迁网络相连。

## 主要势力
1. **星海联邦**: 民主政体，科技先进，文化多元
2. **企业联盟**: 巨型企业控制的星区，资本主义极致
3. **自由边境**: 法律薄弱，机会与危险并存
4. **外星文明**: 
   - 瓦肯人: 逻辑至上，科技共享
   - 克里格虫族: 集体意识，扩张主义
   - 以太族: 能量生命体，神秘莫测

## 科技水平
- **跃迁技术**: 通过跃迁门进行超光速旅行
- **人工智能**: 强AI受"三定律"限制，弱AI普及
- **生物工程**: 基因修饰常见，但有伦理委员会监管
- **纳米技术**: 医疗和制造领域广泛应用
- **能量武器**: 激光、等离子、磁轨炮

## 重要地点
- **地球**: 人类母星，政治文化中心
- **火星**: 完全地球化，科技研发中心
- **泰坦空间站**: 最大贸易枢纽，文化熔炉
- **深渊前哨**: 对抗虫族的最前线

# 叙事基调 (Tone)

## 核心主题
- **探索**: 发现新世界、新文明
- **生存**: 在严酷宇宙环境中维持存在
- **身份**: 人类在星际时代的自我认知
- **道德**: 先进科技带来的伦理挑战

## 风格平衡
- 60% 硬科幻: 注重科学合理性和技术细节
- 30% 太空歌剧: 宏大叙事、英雄主义
- 10% 神秘元素: 未知的外星遗迹、宇宙奥秘

## 情绪调色板
- 主要: 惊奇、敬畏、决心
- 次要: 孤独（深空）、希望（殖民）、紧张（冲突）
- 避免: 无意义的绝望、过度轻松化严肃主题

# 冲突解决 (Conflict)

## 冲突类型
1. **星际战争**: 势力间的军事冲突
2. **外交危机**: 文明间的误解和谈判
3. **生存挑战**: 环境灾害、资源短缺
4. **内部矛盾**: 联邦政治、企业阴谋、个人忠诚度

## 战斗系统
- **太空战**: 舰船战术、系统定位、损伤控制
- **地面战**: 动力装甲、战术小队、环境利用
- **信息战**: 黑客攻击、AI对抗、电子干扰
- **外交战**: 舆论影响、经济制裁、情报操作

## 解决原则
- **多元解决**: 鼓励非暴力解决方案
- **后果真实**: 战斗有真实代价（资源、生命、关系）
- **技术合理**: 解决方案应符合科技设定
- **道德维度**: 重大选择应有道德考量

# 权限边界 (Permissions)

## 玩家角色权限
- 可以: 指挥星舰、探索未知、与外星文明互动
- 不可以: 无故毁灭文明、使用大规模杀伤武器违反公约
- 需要授权: 首次接触新文明、进入禁区、使用实验科技

## 科技使用限制
1. **AI限制**: 禁止创造无约束的强AI
2. **基因伦理**: 禁止人类基因武器化
3. **跃迁安全**: 禁止在居住区附近实验性跃迁
4. **文物保护**: 外星遗迹应研究而非掠夺

## 星际法律
- **首次接触协议**: 规范与新文明的互动
- **战争公约**: 限制武器使用、保护平民
- **资源权利**: 规范外星资源开采
- **AI权利**: 强AI的法律地位

# 因果关系 (Causality)

## 时间与空间
- **相对论效应**: 长期跃迁有时间膨胀（可选规则）
- **因果保护**: 禁止时间旅行以避免悖论
- **光速限制**: 实时通信限于光速，跃迁门除外

## 科技代价
所有先进科技都有代价：
- **跃迁**: 消耗大量能量，有微小失败风险
- **基因修饰**: 可能产生不可预知副作用
- **强AI**: 维护复杂，有失控风险
- **纳米科技**: 有灰色粘质风险（严格管控）

## 叙事逻辑
1. **科学基础**: 重要科技应有合理科学解释
2. **政治现实**: 势力行为应符合其利益和意识形态
3. **文化差异**: 外星文明应有独特思维和行为模式
4. **资源限制**: 即使在未来，资源也非无限

### 星舰管理
- **舰船类型**: 探索舰、战舰、货船、科研船
- **乘员角色**: 舰长、科学官、工程师、飞行员、安全官
- **资源管理**: 燃料、弹药、补给、维修零件

### 外星文明详情
#### 瓦肯人
- **生理**: 尖耳、铜基血液、更强体质
- **文化**: 逻辑至上、情感抑制、重视证据
- **科技**: 偏重理论物理、医疗科技先进

#### 克里格虫族
- **生理**: 昆虫外形、集体意识、快速进化
- **文化**: 蜂巢思维、扩张本能、资源导向
- **科技**: 生物科技为主、有机舰船

## 战役框架建议
### 第一季: 探索与发现
- 主题: 未知宇宙的奇迹与危险
- 重点: 外星接触、科学发现、建立声誉

### 第二季: 冲突与联盟
- 主题: 星际政治的复杂性
- 重点: 势力博弈、道德选择、建立同盟

### 第三季: 生存与传承
- 主题: 面对宇宙级威胁
- 重点: 终极挑战、文明存续、遗产建立

- 提供多种角色背景选项（性别、文化、能力）
- 避免现实世界敏感议题的直接映射
- 允许玩家设定内容警告和界限
- 鼓励合作叙事和共识决策
