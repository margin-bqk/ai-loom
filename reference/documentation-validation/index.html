    术语表¶

    术语 定义 使用位置 一致性
    会话 (Session) 用户与系统的交互过程 所有文档 ✅ 一致
    回合 (Turn) 单次交互循环 所有文档 ✅ 一致
    规则 (Rule) Markdown定义的约束 所有文档 ✅ 一致
    记忆 (Memory) 结构化叙事状态 所有文档 ✅ 一致
    干预 (Intervention) 玩家对叙事的修改 所有文档 ✅ 一致

    建议工具¶

    1. Vale: 文档样式检查
    2. markdown-link-check: 链接检查
    3. markdownlint: Markdown格式检查

    检查脚本示例¶

    #!/bin/bash
# docs_check.sh

# 检查链接
find docs/ -name "*.md" -exec markdown-link-check {} \;

# 检查格式
find docs/ -name "*.md" -exec markdownlint {} \;

# 检查术语一致性

    LOOM 项目文档整体质量良好，核心文档完整且准确。主要需要改进的方面包括：
